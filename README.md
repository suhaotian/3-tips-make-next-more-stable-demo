# Demo code of the article

- Use pnpm monorepo
- `./api-server` for the API server
- `./next-app` for the Next.js example

Install deps:

```sh
pnpm install
```

Run dev:

```sh
pnpm dev
```

Pages:

After run dev,

Open `http://localhost:3000` in browser

Or open `http://localhost:3000/improved` in browser
